> 2025년부터 사람들이 정보를 찾는 방식이 바뀌었습니다. **ChatGPT, Perplexity, Claude, 네이버 큐:** 같은 생성형 검색이 늘면서 기존 SEO만으로는 부족해요. AEO(Answer Engine Optimization)·GEO(Generative Engine Optimization)는 **AI가 우리 사이트 내용을 인용해 답변하도록 만드는 작업**입니다.

## 시작하기 전에

- **개인 도메인 + SSL 적용 완료**: AI 크롤러는 https 사이트만 신뢰합니다.
- **기존 SEO 기초 완료**: [SEO 6단계 체크리스트](#)부터 끝내세요. 그게 안 되면 AEO도 의미 없습니다.
- **소요 시간**: 사이트 전체 점검 1~2시간. 페이지당 콘텐츠 다듬기 별도.

---

## SEO와 AEO의 차이

| 항목 | 기존 SEO | AEO/GEO |
|---|---|---|
| 목표 | 검색 결과 상위 노출 | AI 답변에 인용·요약 |
| 핵심 평가 | 키워드, 백링크, 체류 시간 | **명확한 정답 문장**, 구조화된 정보, 신뢰 신호 |
| 사용자 행동 | 클릭해서 사이트 방문 | 클릭 없이 답변만 받음 |
| 트래픽 영향 | 직접 유입 | 브랜드 노출 + 일부 클릭 |

> 💡 AEO를 잘 해두면 **트래픽이 줄어도 인지도는 올라가고**, 결국 신뢰 기반의 직접 검색(브랜드 검색)이 늘어납니다.

---

## 아임웹에서 AEO 설정 켜기 (2단계)

### 1단계: AI 검색 허용

1. 관리자 페이지 → **환경설정 > SEO**.
2. **AI 검색 허용** 옵션 켜기.
3. 저장.

### 2단계: llms.txt 활성화

`llms.txt`는 AI 크롤러를 위한 새로운 표준 파일로, AI가 우리 사이트의 어떤 콘텐츠를 우선 학습할지 알려줍니다.

1. 같은 화면(**환경설정 > SEO**)에서 **llms.txt 사용** 옵션 켜기.
2. 저장.

> 📌 두 설정 모두 켜야 ChatGPT의 SearchGPT, Perplexity, Claude의 검색 모드 등에 정확히 노출됩니다.

## 콘텐츠 최적화 5가지

### 1. 페이지 최상단에 "40~60단어 핵심 요약"

AI는 페이지의 첫 부분을 가장 중요하게 봅니다. **"누구를 위한, 어떤 정보·서비스인지"**를 첫 2~3문장 안에 명확히.

❌ 안 좋은 예:
> 안녕하세요. 저희는 1990년부터 이 자리를 지켜온 가족 같은 병원입니다. 늘 환자분의 마음을 헤아리며…

✅ 좋은 예:
> 안산 본오동에 위치한 ○○이비인후과는 코·귀·목 전문 진료를 제공합니다. 평일은 야간 9시까지, 토요일도 진료. 수면다원검사·중이염·알레르기 비염을 주로 다룹니다.

### 2. 구조화된 정보 (수치·규격 명시)

모호한 형용사보다 **구체적인 수치**가 인용 가능성을 높입니다.

| 모호함 | 구체적 |
|---|---|
| 빠른 응답 | 평균 응답 시간 2시간 이내 (평일 영업시간) |
| 합리적인 가격 | 평일 비급여 진료비 2~5만원대 |
| 다양한 경험 | 2015년 개원, 누적 진료 12,000건 |

### 3. FAQ 섹션 필수

**자주 묻는 질문 5~10개를 간결한 답변(2~4문장)과 함께** 페이지마다 운영.

AI는 FAQ 구조를 매우 좋아합니다. "Q. ... / A. ..." 형식이 인용 가능성을 압도적으로 높여요.

> 💡 FAQPage schema를 추가하면 더 강력. 아임웹은 자동 생성하지 않으므로 **환경설정 > SEO > 고급 설정 > Header Code**에 직접 JSON-LD 삽입.

### 4. 고객 리뷰·후기 적극 활용

리뷰는 **신뢰 신호**의 핵심. AI는 "검증된 사이트"를 우선 인용합니다.

- 사진 + 텍스트 리뷰 (텍스트만 있으면 신뢰도 ↓)
- 사용 상황·결과 포함 ("3개월 사용 후 코막힘 70% 개선")
- 별점 + 자유 후기 병행

### 5. 메타 설명을 "답변 형태"로

기존 SEO는 키워드 위주, AEO는 **질문에 대한 답**.

❌ `안산 이비인후과, 본오동 병원, 코 알레르기`
✅ `안산 본오동에서 야간·토요일 진료 가능한 이비인후과를 찾는다면 ○○이비인후과. 수면다원검사부터 비염까지 한 곳에서 진료.`

## 페이지 유형별 AEO 체크리스트

### 회사·병원 소개 페이지
- [ ] 한 문장 정의 ("○○은 ~를 제공하는 ~입니다")
- [ ] 위치·연락처·운영시간 명시 (Local SEO 겸용)
- [ ] 대표 서비스 5~7개를 불릿으로 나열
- [ ] FAQ 5개 이상

### 서비스·진료 안내 페이지
- [ ] 어떤 증상·상황에 적합한지 명시
- [ ] 절차를 단계별로 (1, 2, 3...)
- [ ] 소요 시간·비용 범위
- [ ] 자주 묻는 질문 + 답

### 블로그·뉴스 페이지
- [ ] 첫 문단에 결론 먼저 (TL;DR)
- [ ] H2 헤더를 **질문 형태로**: "○○이란?", "왜 ○○해야 하나?"
- [ ] 표·리스트 적극 사용 (AI가 구조 인식하기 좋음)
- [ ] 인용·출처 표기

## llms.txt 직접 작성 (고급)

아임웹은 기본 `llms.txt`를 자동 생성하지만, 직접 작성하면 더 정밀한 제어가 가능합니다.

루트에 `llms.txt`를 두면 AI는 이것부터 읽습니다.

**기본 구조 예시**:
```markdown
# ○○이비인후과

> 안산 본오동의 코·귀·목 전문 이비인후과. 야간·토요일 진료.

## 핵심 서비스
- [수면다원검사](https://mycompany.com/sleep-test): 코골이·수면무호흡증 진단
- [알레르기 비염 클리닉](https://mycompany.com/allergy): 검사+면역요법
- [중이염 진료](https://mycompany.com/otitis): 성인·소아 모두

## 자주 묻는 질문
- 진료 예약: https://mycompany.com/reservation
- 비용 안내: https://mycompany.com/pricing
- 위치·교통: https://mycompany.com/location

## 추가 자료
- 블로그: https://mycompany.com/blog
```

## 효과 확인하는 법

### 즉시 확인 (2주~1개월)

- **ChatGPT**: "안산 본오동 이비인후과 추천해줘" 검색 → 우리 사이트가 답변에 인용되는지.
- **Perplexity**: 동일 검색 후 출처 목록에 우리 사이트 있는지.
- **네이버 큐:**: 베타 단계지만 답변 소스에 노출되는지.

### 장기 확인 (3개월~)

- Google Search Console > **수행 도구** > Discover·SearchGPT 트래픽 (있다면)
- 브랜드 검색 증가율 (`mycompany 이름` 직접 검색이 늘어나는지)

## ⚠️ 자주 막히는 포인트

- **AI 허용·llms.txt 둘 다 안 켬** → 가장 흔한 실수. 두 옵션 모두 ON 필수.
- **콘텐츠가 광고 카피로만 도배됨** → AI는 정보성 콘텐츠를 우선 인용. 마케팅 문구만 있으면 무시됨.
- **FAQ 없음** → AEO 핵심 누락.
- **수치 없는 모호한 표현** → "다양한 경험" 같은 형용사는 AI에게 무의미.
- **이미지 안에만 정보가 있음** → AI는 이미지 텍스트를 잘 읽지 못함. 텍스트로도 같이.

## ❓ FAQ

**Q. SEO를 안 했는데 AEO부터 해도 되나요?**
A. 권장하지 않습니다. AEO는 SEO 위에 얹는 작업이에요. 사이트맵 제출·메타 정보 같은 기초가 없으면 AI 크롤러도 사이트를 발견하지 못합니다. [SEO 기초 6단계](#) 먼저.

**Q. AEO 하면 트래픽이 늘어요?**
A. 단기적으로는 오히려 줄 수 있습니다. AI가 답변을 직접 주면 사용자가 클릭을 안 하니까요. 다만 **브랜드 인지도·신뢰도**가 올라가서, 결과적으로 직접 검색·전화 문의가 늘어나는 경향이 있습니다.

**Q. ChatGPT에 안 보이는데요?**
A. ChatGPT가 우리 사이트를 학습·색인하는 데도 시간이 걸립니다(약 1~3개월). 또한 AI 검색 결과는 사용자마다 다르게 나옵니다. 시크릿창·다른 계정에서도 확인해 보세요.

**Q. llms.txt를 안 만들어도 되나요?**
A. 아임웹이 자동 생성하므로 안 만들어도 기본은 됩니다. 다만 사이트 구조가 특수하거나, 특정 페이지를 강조하고 싶다면 직접 작성을 권장합니다.

**Q. 경쟁 업체가 우리 콘텐츠 베껴서 AI에 노출되면요?**
A. AI는 **첫 출처·도메인 권위·일관성**을 중요시합니다. 일관된 콘텐츠 업데이트와 외부 인용·언급이 쌓이면 카피본보다 원본이 우선 노출됩니다.
