> 아임웹은 코딩 없이 사이트를 만들고 운영할 수 있지만, 막상 운영하다 보면 도메인 연결·메일 설정·디자인 수정·검색 노출처럼 막히는 지점이 생깁니다. 이 글은 가이드센터의 아임웹 글 전체를 **운영 흐름 순서대로** 안내하는 지도입니다.

## 운영 흐름 한눈에

```
시작 → 도메인·메일 연결 → 메뉴·페이지 구성 → 디자인 다듬기
→ 폼·회원 → 검색 노출 → 마케팅·통계
```

## ⚠️ 운영하며 자주 막히는 순서

- **도메인 연결 후 SSL 미적용**: https가 안 떠 경고 표시 → SSL 신청 필수
- **메일 설정 없이 폼 운영**: 문의가 와도 알림 메일이 안 감
- **PC만 보고 모바일 방치**: 방문의 다수가 모바일
- **검색 등록 누락**: 사이트는 있는데 검색에 안 뜸

## ❓ FAQ

**Q. 아임웹과 WordPress 중 뭐가 나은가요?**
A. 아임웹은 쉽고 빠르게 운영, WordPress는 자유도가 높습니다. 직접 간편하게 운영하려면 아임웹, 확장·커스터마이징이 중요하면 WordPress예요. ([WordPress 가이드](/guides/wp-overview/) 참고)

**Q. 아임웹도 SEO가 되나요?**
A. 됩니다. 다만 자체 인프라라 일부 기술 항목은 통제가 제한적이에요. 기본 SEO 설정([SEO 6단계](/guides/imweb-seo-basics/))을 충실히 하고, 콘텐츠 품질로 승부하는 게 좋습니다.

**Q. 어디부터 시작해야 하나요?**
A. 처음 시작하기 → 도메인·SSL → 메뉴 구성 → 디자인 순서를 권합니다. 사이트가 보이는 형태를 먼저 갖추고, 그 다음 SEO·마케팅으로 확장하세요.
