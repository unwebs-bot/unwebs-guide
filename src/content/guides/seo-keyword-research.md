> 키워드 리서치는 SEO의 출발점입니다. **사람들이 실제로 검색하는 단어**를 찾아 그중 우리에게 유리한 키워드부터 공략하는 작업이에요. 도구 5개·검색 의도 4분류·우선순위 매트릭스까지 정리합니다.

## 키워드 리서치를 왜 해야 할까

콘텐츠를 아무리 잘 써도 **아무도 검색하지 않는 단어**로 작성하면 노출이 안 됩니다. 반대로 검색량 많은 키워드만 노리면 거대 사이트와의 경쟁에서 묻힙니다.

세 가지 균형:
- **검색량**: 얼마나 많이 찾는가
- **경쟁도**: 상위 노출이 얼마나 어려운가
- **전환 가능성**: 검색한 사람이 우리 고객이 될 확률

## 도구 5개

### 1. 네이버 키워드 도구 (무료)

[searchad.naver.com](https://searchad.naver.com) → **도구 > 키워드 도구**

- 네이버 검색량 정확
- 연관 키워드 자동 제안
- 월별 검색량 추이
- 모바일·PC 분리

> 한국 시장 SEO를 한다면 1순위. 무료지만 광고주 계정 가입 필요.

### 2. Google Keyword Planner (무료)

[ads.google.com](https://ads.google.com) → 도구 > 키워드 플래너

- 구글 검색량
- 키워드 아이디어 발굴
- 입찰가(키워드별 광고 단가) 확인

> Google Ads 계정 필요. 광고 안 돌려도 가입만 하면 사용 가능.

### 3. Google Trends (무료)

[trends.google.com](https://trends.google.com)

- 시간대별 관심도 추이
- 지역별 인기 비교
- 관련 급상승 키워드

> "지금 뜨는 키워드" 발견에 강함. 시즌성 콘텐츠 기획에 필수.

### 4. AnswerThePublic (무료/유료)

[answerthepublic.com](https://answerthepublic.com)

- 키워드 + Who/What/When/Where/Why 자동 조합
- 사람들이 실제로 던지는 **질문 형태** 키워드
- 무료 1일 3회 제한

> 블로그·FAQ 콘텐츠 아이디어에 최강.

### 5. 경쟁사 분석 — Ahrefs / SEMrush (유료)

- 경쟁사가 어떤 키워드로 유입되는지
- 백링크 분석
- 콘텐츠 갭 분석

> 본격 SEO 운영하면 월 $99~ 가치 충분. 1~2개월 단기 구독으로 데이터 한번 뽑아도 OK.

## 검색 의도 4분류

키워드를 검색할 때 사용자가 **원하는 결과**가 다릅니다.

| 의도 | 예시 키워드 | 검색자 상태 | 콘텐츠 형식 |
|---|---|---|---|
| **Informational** (정보) | "SEO란", "도메인 연결 방법" | 학습 중 | 가이드·블로그·튜토리얼 |
| **Navigational** (탐색) | "언웹스", "imweb 로그인" | 특정 사이트로 가려는 중 | 브랜드 페이지·로그인 |
| **Commercial** (조사) | "이비인후과 추천 안산", "WP 호스팅 비교" | 구매 전 비교 | 비교표·리뷰·추천 글 |
| **Transactional** (구매) | "안산 본오동 이비인후과 예약", "도메인 구매" | 결정 직전 | 랜딩 페이지·예약 폼 |

> 💡 **같은 사이트라도 페이지마다 의도가 달라야** 합니다. 메인은 Navigational, 블로그는 Informational, 가격 페이지는 Commercial, 예약 페이지는 Transactional.

## 우선순위 매트릭스

키워드 후보 리스트가 100개라면 어떻게 줄일까?

### 4분면 매트릭스

```
            검색량 高
              │
   "황금 영역"  │  "거대 경쟁"
   상위 가능     │  대기업 광고
              │
  경쟁 ─────────┼───────── 경쟁
   低         │          高
              │
   "조용한 자산"│  "버리는 영역"
   롱테일      │  비용 高 효율 低
              │
            검색량 低
```

**우선 공략**: 검색량 中 + 경쟁도 低 = **"황금 영역"**

### 점수 매기기

각 키워드에 1~5점:
- 검색량 점수 (1=거의 없음, 5=월 1만+)
- 경쟁도 역수 (1=대기업 도배, 5=경쟁 거의 없음)
- 전환 관련성 (1=관련 약함, 5=직접 고객층)

**총점 = 검색량 + 경쟁역수 + 전환관련성** (최대 15점). 12점 이상부터 공략.

### 예시 (병원 사이트)

| 키워드 | 검색량 | 경쟁역수 | 전환관련성 | 총점 |
|---|---|---|---|---|
| 이비인후과 | 5 | 1 | 2 | 8 ❌ |
| 안산 이비인후과 | 4 | 3 | 5 | 12 ✅ |
| 본오동 이비인후과 | 2 | 5 | 5 | 12 ✅ |
| 코 알레르기 야간 진료 안산 | 1 | 5 | 5 | 11 ⚠️ |
| 이비인후과 예약 | 4 | 2 | 4 | 10 |

**전략**:
- "안산 이비인후과" 메인 페이지 SEO 타겟
- "본오동 이비인후과" 별도 랜딩
- "코 알레르기 야간 진료 안산" 블로그 글로 롱테일

## 롱테일 키워드 — 작지만 강함

3단어 이상의 구체적 키워드를 롱테일이라 부릅니다.

| 짧은 키워드 | 롱테일 |
|---|---|
| 이비인후과 | 안산 본오동 야간 진료 이비인후과 |
| 홈페이지 제작 | 안산 1인 웹팩토리 병원 홈페이지 제작 |
| WordPress | WordPress 백업 자동화 무료 플러그인 |

특징:
- 검색량 적음 (월 10~100건)
- 경쟁 거의 없음
- **전환율 압도적으로 높음** (구체적이라 결정 직전 검색)
- 50개 만들면 합산 트래픽이 짧은 키워드 1개 못지않음

> 💡 신생 사이트는 **롱테일부터** 시작. 짧은 키워드는 도메인 권위가 쌓인 후.

## 키워드 그룹화 = 페이지 설계

100개 키워드를 발견했다면 그대로 100페이지를 만들면 안 됩니다. **의도가 같은 키워드는 한 페이지에**.

예시 그룹:
- `안산 이비인후과` + `안산 이비인후과 추천` + `안산 본오동 이비인후과` → 1개 메인 페이지
- `수면다원검사 비용` + `수면다원검사 보험` → 1개 정보 페이지
- `이비인후과 예약 방법` → 1개 예약 안내 페이지

## 키워드 발굴 워크플로우 (실제 작업)

### 1주차

1. **시드 키워드 5~10개** 추리기 (브랜드명·핵심 서비스명)
2. 네이버 키워드 도구에 시드 입력 → 연관 키워드 다운로드
3. AnswerThePublic으로 질문형 키워드 발굴
4. Google Trends로 시즌성 확인

### 2주차

5. 경쟁사 사이트 5곳 골라 어떤 키워드로 유입되는지 추정 (Ahrefs 무료 체험)
6. 모든 키워드를 스프레드시트에 정리 → 점수 매기기
7. 12점 이상 키워드 30개 선별
8. 의도별로 그룹화 → 페이지 설계

### 3주차~

9. 우선순위 1~5 키워드 콘텐츠 작성
10. Search Console에서 색인 요청
11. 1개월 후 검색 노출 데이터로 다음 우선순위 조정

## ⚠️ 자주 하는 실수

- **검색량만 보고 키워드 결정** → 경쟁도·전환관련성 같이 봐야 함.
- **모든 키워드를 개별 페이지로** → 의도 같은 키워드는 한 페이지로 통합.
- **시즌성 키워드 무시** → 봄·여름 키워드 분리 운영하면 트래픽 2배.
- **본인이 쓰는 단어 = 검색자가 쓰는 단어 착각** → 전문 용어 vs 일반 용어 차이. 사용자 시각으로 검색해야.
- **키워드 박제** → 한 페이지에 키워드 20개 우겨넣기. 자연스러운 1~3개 위주.

## ❓ FAQ

**Q. 무료 도구만으로 충분한가요?**
A. 시작 단계는 충분. 네이버 키워드 도구 + Google Keyword Planner + Google Trends 조합이면 70% 커버. 본격 운영 시 Ahrefs/SEMrush 단기 구독으로 보완.

**Q. 키워드 검색량 0이면 안 만드는 게 좋나요?**
A. 검색량 0이지만 **전환 직결 키워드**(예: "OO병원 예약 취소")는 만들어야 합니다. 검색자가 적어도 100% 우리 고객.

**Q. 영어·외국어 키워드는 어떻게?**
A. Google Keyword Planner가 다국어 지원. 다만 한국 비즈니스라면 한국어 우선.

**Q. 키워드 한 페이지에 몇 개 넣어야 하나요?**
A. **타겟 키워드 1개**, 보조 키워드 3~5개, 변형 표현(동의어) 자연스럽게. 키워드 밀도 1~2%면 충분.

**Q. 키워드 리서치 얼마나 자주 해야?**
A. 신규 콘텐츠 기획 시마다. 전체 재점검은 분기 1회. 트렌드 빠른 업종(IT·뷰티)은 월 1회.
