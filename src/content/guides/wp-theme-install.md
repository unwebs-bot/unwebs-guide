> 테마는 WordPress 사이트의 **디자인·구조 뼈대**입니다. 잘 고르면 운영이 편하고 빠르고, 잘못 고르면 무겁고 충돌이 잦아 사이트가 망가져요. 테마 선택 기준·설치 절차·전환 시 주의사항을 정리.

## 테마 선택 5가지 기준

### 1. 가벼움 (속도)

테마 파일 크기·로드되는 CSS/JS 양이 사이트 속도에 결정적.

**가벼운 테마 추천**:
- **GeneratePress** — 가장 가벼움 (10KB 미만 CSS)
- **Astra** — 인기 1위, 가벼움
- **Kadence** — 빠름 + 디자인 좋음
- **Blocksy** — 모던 + 빠름

**무거운 테마 (주의)**:
- 시각 빌더 내장 (Avada·Divi·Bridge) — 화려하지만 무거움
- 다목적 테마 (모든 산업 지원) — 사용 안 하는 코드 많음

### 2. 호환성

자주 쓰는 플러그인과 충돌 없는지:
- WooCommerce (쇼핑몰)
- Elementor·Gutenberg
- Rank Math·Yoast SEO
- 인기 폼·캐시 플러그인

### 3. 업데이트 지속

마지막 업데이트 날짜 확인. 1년 넘게 업데이트 없는 테마는 보안·호환성 위험.

### 4. 디자인 자유도

- 색상·폰트 자유 변경
- 헤더·푸터 옵션 풍부
- 자식 테마 지원

### 5. 한국어·CJK 폰트 지원

한글 폰트 깨지지 않는지. 대부분 테마가 OK지만 일부 테마가 한글 자간·줄간격이 이상함.

## 무료 vs 유료

| | 무료 | 유료 (Premium) |
|---|---|---|
| 가격 | 0 | $30~$100/년 |
| 디자인 | 제한적 | 풍부 |
| 지원 | 커뮤니티 | 공식 지원 |
| 업데이트 | 비정기 | 정기 |
| 자식 테마 | 일부 지원 | 대부분 지원 |

**추천**:
- 입문·블로그: 무료로 충분
- 비즈니스 사이트: Pro 버전 (Astra Pro, GeneratePress Premium 등)

## 안전하지 않은 무료 테마 — 조심

- **Nulled (무단 배포) 프리미엄**: 악성코드 90% 확률
- **출처 불명 사이트**: 검증 안 됨
- **테마 사이트라며 ZIP 다운로드 강요**: 의심

**안전한 출처**:
- **WordPress.org** 공식 저장소 (모두 검증)
- **테마 제작사 공식 사이트** (Astra·GeneratePress·Elegant Themes 등)
- **ThemeForest** (유료, 검수됨)

## 테마 설치 — 2가지 방법

### 방법 1: 공식 저장소

1. **외모 > 테마 > 새로 추가**
2. 검색 → 테마 클릭 → **설치**
3. **활성화**

### 방법 2: ZIP 업로드

유료 테마·외부 다운로드 시:

1. **외모 > 테마 > 새로 추가 > 테마 업로드**
2. ZIP 파일 선택 → **지금 설치**
3. **활성화**

## 테마 전환 시 안전 절차 — 7단계

운영 중인 사이트의 테마를 바꾸는 건 큰 작업. 다음 순서로:

### 1. 전체 백업

[백업 시작하기](/guides/wp-backup-basics/) — UpdraftPlus로 풀 백업.

### 2. 현재 상태 기록

- 메뉴 위치
- 위젯 영역과 위젯
- 커스터마이저 설정 값 (스크린샷)
- 자식 테마 코드

### 3. 스테이징 환경에서 테스트

옵션 A: Local by Flywheel에 복제 → 새 테마 테스트
옵션 B: 호스팅의 staging 기능 (있다면)
옵션 C: 서브도메인에 임시 설치

새 테마 활성화 → 사이트 점검 → 문제없으면 본 사이트 적용.

### 4. 새 테마 설치 (운영 사이트)

위 절차로 설치만 (활성화 아직 X).

### 5. 유지보수 모드

플러그인 **WP Maintenance** — 방문자에게 "공사중" 표시.

### 6. 활성화 → 즉시 점검

- 메뉴 재지정 (테마마다 메뉴 위치 다름)
- 위젯 영역 재배치
- 커스터마이저 설정 다시
- 모든 페이지 깨짐 확인

### 7. 유지보수 모드 해제

문제없으면 공개.

## 자주 깨지는 곳

테마 전환 시:

- **메뉴**: 위치 매핑 다시
- **위젯**: 영역 이름이 다르면 위젯이 비활성 영역에 들어감
- **커스텀 디자인 코드**: 옛 테마용이라 작동 X
- **단축코드**: 테마가 정의한 건 안 작동
- **이미지 크기**: 자동 크기가 달라 깨짐
- **폰트**: 옛 테마의 구글 폰트가 안 로드됨

## 자주 받는 질문

### Q. 어떤 테마를 골라야 할지 모르겠어요.

**일반 비즈니스**: GeneratePress 또는 Astra (무료로 시작)
**디자인 좋은 사이트**: Kadence·Blocksy
**쇼핑몰**: Astra·OceanWP (WooCommerce 친화)
**저널·매거진**: Newspaper·Soledad
**일단 시작**: WordPress 기본 테마 (Twenty Twenty-Five)

### Q. 테마 가격 $50 정도면 비싼가요?

연 $30~80은 표준. 매년 갱신은 선택 (코드는 그대로 사용 가능, 업데이트만 X).

### Q. 테마 자체 페이지 빌더 vs Elementor?

테마 빌더 (Avada Builder·Divi)는 테마 종속. 다른 테마 전환 시 다 깨짐.
**Elementor·Gutenberg**가 더 자유. 권장.

### Q. 테마를 직접 만들 수 있나요?

가능. PHP·HTML·CSS·JS 필요. 입문자는 자식 테마 커스텀이 현실적. [자식 테마 가이드](/guides/wp-child-theme/) 참고.

### Q. 테마와 플러그인 차이?

- **테마**: 사이트 디자인·구조
- **플러그인**: 기능 추가

겹치는 영역도 있음 (예: SEO 플러그인 vs 테마 SEO 기능).

## ⚠️ 자주 하는 실수

- **유명 다목적 테마** 무조건 선택: 무겁고 자유도 ↑가 아닌 ↓
- **테마 자체 페이지 빌더 종속**: 테마 변경 시 다 깨짐
- **자식 테마 안 만들고 직접 코드 수정**: 업데이트 시 다 사라짐
- **무료라고 출처 불명 테마 다운로드**: 악성코드
- **운영 중 갑자기 테마 전환**: 백업·테스트 없이 → 사이트 망가짐

## ❓ FAQ

**Q. 테마 갱신 결제 안 하면 사용 못 하나요?**
A. 사용은 가능 (코드는 본인 것). 다만 업데이트·지원 X. 보안 패치 못 받음.

**Q. 테마를 두 개 설치하면 안 되나요?**
A. 설치는 가능 (활성 1개·비활성 N개). 다만 비활성 테마도 보안 취약점 있을 수 있어 안 쓰는 건 삭제 권장.

**Q. 테마 활성화하니 사이트가 흰 화면이에요.**
A. 테마 PHP 오류. FTP로 `wp-content/themes/` 폴더에서 옛 테마 폴더 복원 또는 다른 이름으로 변경 → WordPress 기본 테마로 fallback. [흰화면 디버그](/guides/wp-white-screen-debug/) 참고.

**Q. 테마와 플러그인 어디까지가 적정선?**
A. 디자인은 테마, 기능은 플러그인. 다만 SEO·폼·캐시 같은 핵심 기능은 플러그인이 강함.

**Q. 한국에서 만든 WordPress 테마가 있나요?**
A. 일부 있지만 글로벌 테마 (Astra·Kadence) 대비 업데이트·지원 약함. 글로벌 테마 + 한글 폰트 적용이 정답.
