> SEO는 한 번에 끝나는 게 아니라 **지속 운영**이 답. 매주·매월·분기별 할 일을 체크리스트로 만들어두면 잊지 않고 운영 가능합니다. WordPress 사이트의 SEO 점검 체크리스트.

## 일회성 — 처음 한 번만

### 셋업 단계 (1주차)

- [ ] WordPress 설치·기본 설정
- [ ] **SSL** 적용 ([SSL 가이드](/guides/wp-ssl-setup/))
- [ ] **퍼머링크** = "글 이름" ([퍼머링크 가이드](/guides/wp-permalink-structure/))
- [ ] 사이트 제목·태그라인 입력
- [ ] **Rank Math SEO** 또는 Yoast 설치·설정 마법사
- [ ] **사이트맵** 자동 생성 확인 (`/sitemap_index.xml`)
- [ ] **Google Search Console** 등록·소유 확인·사이트맵 제출
- [ ] **네이버 서치어드바이저** 등록·사이트맵·RSS 제출
- [ ] **Bing Webmaster Tools** 등록 (Search Console 데이터 임포트)
- [ ] **Google Analytics 4** 설치
- [ ] 본인 IP 제외 (GA4)
- [ ] **robots.txt** 작성 확인
- [ ] 모든 페이지 **모바일 친화도** Mobile-Friendly Test
- [ ] **Core Web Vitals** PageSpeed Insights 첫 측정

### 콘텐츠 준비

- [ ] **About 페이지** 작성 (E-E-A-T 신호)
- [ ] **연락처 페이지** 작성 (Local SEO)
- [ ] **개인정보 처리방침** 작성
- [ ] 회사·기관 정보 (사업자 등록·주소·전화) 푸터에 명시
- [ ] **LocalBusiness Schema** 적용 (지역 비즈니스인 경우)
- [ ] 카테고리·태그 구조 설계

## 주간 (5분)

### 콘텐츠 발행

- [ ] 새 글 1편+ 발행 (가능한 주기)
- [ ] 발행 후 Search Console **URL 검사 > 색인 요청**
- [ ] 신규 글 카테고리·태그·대표 이미지·메타 정보 입력

### 모니터링

- [ ] Search Console 실적 대시보드 (5분)
- [ ] 색인 신규 오류 확인
- [ ] 댓글·문의 응답

---

## 매월 (1시간)

### 데이터 분석

- [ ] **Search Console** 월간 비교
  - 클릭 추이
  - 노출 추이
  - 평균 순위 변동
- [ ] **GA4** 월간 비교
  - 활성 사용자
  - 세션 소스/매체
  - 인기 페이지
  - 모바일 vs PC 비율

### 콘텐츠 최적화

- [ ] **CTR 낮은 페이지 5개** 개선 (title·description)
- [ ] **11~20위 키워드** 부스트할 페이지 보강
- [ ] **404 페이지** 정리 (301 또는 삭제)
- [ ] 옛 글 1~2편 업데이트 (정보 갱신)

### 기술 점검

- [ ] WordPress 코어·플러그인·테마 업데이트
- [ ] 보안 플러그인 스캔
- [ ] 백업 작동 확인
- [ ] Core Web Vitals 점수
- [ ] 사이트 헬스 점수

### 이미지 정리

- [ ] 사용 안 한 미디어 정리
- [ ] 큰 이미지 압축 (ShortPixel 일괄)
- [ ] ALT 텍스트 누락 점검

## 분기 1회 (반나절)

### 콘텐츠 대점검

- [ ] **전체 콘텐츠 성과 분석**
  - 트래픽 상위 10편
  - 트래픽 0인 글 (통합·삭제·재작성)
  - 옛 글 갱신 우선순위
- [ ] **카테고리·태그 정리**
  - 사용 안 한 태그 삭제
  - 비슷한 태그 통합
  - 카테고리 구조 재검토
- [ ] **내부 링크 강화**
  - 핵심 페이지로의 링크 5개+
  - 옛 글에 신규 글 링크 추가

### 외부 환경

- [ ] **백링크 점검** (Search Console > 링크)
- [ ] **부정 백링크** 발견 시 Disavow
- [ ] **경쟁사 분석** 1곳 (Ahrefs 무료)
- [ ] **순위 추이** 핵심 키워드 (Rank Math 또는 외부 도구)

### 기술 점검 (깊이)

- [ ] **PHP 버전** 최신 검토 (8.0+)
- [ ] **호스팅** 응답 속도 (TTFB)
- [ ] **DB 최적화** (WP-Optimize)
- [ ] **사이트 속도** 전체 페이지 측정
- [ ] **모바일 친화도** 모든 페이지
- [ ] **HTTPS 일관성** (혼합 콘텐츠 없음)
- [ ] **canonical URL** 설정 점검

### 콘텐츠 캘린더

- [ ] 다음 분기 콘텐츠 주제 10~20개 기획
- [ ] 시즌·트렌드 콘텐츠 반영

## ⚠️ 자주 하는 실수

- **체크리스트만 만들고 운영 X**: 캘린더에 반복 일정 등록
- **너무 많은 도구**: 핵심 6개만으로 충분
- **단기 변동 과민 반응**: 한 달 추이로 판단
- **데이터 안 보고 직감**: GA·GSC가 진실
- **콘텐츠보다 기술에 집착**: 80%는 콘텐츠가 결정

## ❓ FAQ

**Q. 모든 체크리스트를 다 따라야 하나요?**
A. 사이트 규모·운영 자원에 맞게. 작은 사이트는 매주 5분 + 매월 30분 정도면 충분.

**Q. 분기 점검에 반나절이 부담스러워요.**
A. 한 번에 다 안 해도 OK. 매주 5분 점검에서 일부 분기 항목 분산.

**Q. SEO 회사에 맡기는 게 좋나요?**
A. 기초·매주 점검은 본인. 백링크·콘텐츠 양산은 부분 외주 OK. 외주만 의존은 위험.

**Q. 효과 측정은 언제?**
A. SEO 작업은 1~3개월 후 효과. 데이터 추이로 판단. 일주일 변동은 정상.

**Q. 어디서부터 시작해야 할지 모르겠어요.**
A. 일회성 셋업 → 주간 발행 + 모니터링. 3개월 후 매월 점검 추가.
