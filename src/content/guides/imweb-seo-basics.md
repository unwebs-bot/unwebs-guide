> 아임웹으로 만든 사이트가 네이버·구글에 노출되려면 **사이트 등록 → 메타 정보 → 사이트맵 제출 → 페이지별 최적화** 순서로 챙기면 됩니다. 이 글 하나만 끝까지 따라 하면 검색 노출 기초 세팅이 완료됩니다.

## 시작하기 전에

- **요금제 확인**: Free 버전은 검색엔진 색인이 제한적입니다. Starter 이상 권장.
- **개인 도메인 연결 완료**: 검색엔진은 `*.imweb.me`보다 개인 도메인을 신뢰합니다. 도메인 미연결 시 [내 도메인 연결하는 법](#) 먼저.
- **SSL 적용 완료**: 미적용 시 네이버 서치어드바이저 등록부터 막힙니다.
- **소요 시간**: 전체 약 1시간 (계정 보유 시). 검색 결과 반영까지 **약 2~4주** 소요.

## 1단계 — 검색엔진에 사이트 등록하기

### 네이버

가장 우선순위. 한국 검색 트래픽의 60% 이상이 네이버.

1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속 후 로그인.
2. **웹마스터 도구 > 사이트 추가** → 도메인 입력(`https://` 포함).
3. **HTML 태그** 방식으로 소유 확인 메타태그 복사.
4. 아임웹 관리자 → **환경설정 > SEO > 고급 설정**의 **Header Code** 상단에 메타태그 붙여넣고 저장.

![네이버 서치어드바이저 사이트 추가 화면](/imweb/6857658addbf8.png)

5. 서치어드바이저로 돌아가 **확인** 버튼.
6. **자동 등록 방지 문자** 입력 → 등록 완료.

> ⚠️ 네이버 정보 수집까지 **14~16일** 소요. 등록만 한다고 바로 검색되는 게 아닙니다.

### 구글

1. [Google Search Console](https://search.google.com/search-console) 접속 → **시작하기**.
2. **URL 접두어 유형**에 도메인 입력 (`https://mycompany.com`).
3. **HTML 태그** 선택 → 메타태그 복사.
4. 아임웹 SEO 고급 설정의 Header Code 상단에 추가 → 저장.

5. Search Console에서 **확인** → 완료.

> 구글은 등록 후 **2~4주** 이내 색인. 새 페이지를 추가하면 Search Console에서 **URL 검사 > 색인 생성 요청**으로 가속 가능.

### 다음·야후·Bing

선택 사항. 트래픽 비중은 작지만 5분이면 끝나니 같이 하는 걸 권장.

## 2단계 — 메타 정보(Title·Description) 설정

검색 결과에 노출되는 **제목과 설명**. 가장 중요한 SEO 요소.

### 위치

아임웹 관리자 → **환경설정 > 일반**

| 항목 | 권장 글자수 | 예시 |
|---|---|---|
| 사이트 이름 | 10~25자 | `편한이비인후과 - 안산 본오동 진료 전문` |
| 사이트 설명 | **60~80자** | `안산 본오동 편한이비인후과는 코·귀·목 진료부터 수면다원검사까지 제공합니다. 평일 야간·토요일 진료.` |

> 💡 **포인트**: 사이트 설명에는 **"지역 + 업종 + 차별점"**을 자연스럽게 녹이세요. 키워드 나열만 하면 검색엔진 품질평가에 불리합니다.

### 메타 키워드는?

`환경설정 > SEO > 메타 키워드` 항목이 있지만, **거의 영향 없습니다**. 구글은 이미 2009년부터 무시, 네이버도 비중이 매우 작아요. 5~10개 정도만 자연스럽게.

## 3단계 — 사이트맵·RSS 제출

검색 로봇이 사이트 전체 페이지를 빠르게 발견하도록 돕는 단계.

### 사이트맵 주소 확인

아임웹 관리자 → **환경설정 > SEO > 고급 설정** → **사이트맵(sitemap.xml)** 항목에서 주소 복사.

대개 `https://mycompany.com/sitemap.xml` 형식입니다.

### 구글에 제출

1. Search Console → **색인 > Sitemaps**.
2. `sitemap.xml` 입력 → **제출**.
3. 몇 시간 ~ 며칠 후 처리 결과 표시.

### 네이버에 제출

1. 서치어드바이저 → 등록한 사이트 선택.
2. **요청 > 사이트맵 제출** → 복사한 주소 붙여넣기 → **확인**.
3. **요청 > RSS 제출**도 같이. RSS는 새 글 색인 속도를 빠르게 합니다.

![네이버 서치어드바이저 사이트맵 제출](/imweb/3c80f4f183622.png)

> 💡 사이트맵 미제출이어도 검색은 됩니다. 다만 **내부 링크가 없는 페이지**(예: 캠페인 랜딩)는 제출해야 발견됩니다.

## 4단계 — 페이지별 제목·설명 (가장 중요)

각 메뉴(페이지)마다 따로 SEO 정보를 넣어야 검색에서 강해집니다.

### 설정 방법

1. **디자인 모드** 접속.
2. 좌측 상단 **메뉴 관리(≡)** 아이콘 클릭.
3. 수정할 메뉴 옆 **ⓘ 아이콘** 클릭 → **메뉴 설정** 창.
4. **페이지 제목**과 **페이지 설명** 입력.
5. 닫기(✕) → 자동 저장.

![메뉴별 페이지 제목·설명 설정](/imweb/9b196b26aadac.png)

### 권장 글자수 & 작성 팁

| 항목 | 권장 | 작성 팁 |
|---|---|---|
| 페이지 제목 | **15자 이하** | "병원 소개", "진료 안내" 같은 일반 단어 X. "본오동 코 알레르기 진료 - 편한이비인후과" 식. |
| 페이지 설명 | **60~80자** | 그 페이지에서 얻을 수 있는 정보를 구체적으로. |

### 자주 하는 실수

- ❌ 모든 페이지 제목이 똑같음 (사이트 이름만)
- ❌ 페이지 설명이 빈칸 (검색엔진이 자동으로 본문 일부 발췌 → 부정확)
- ❌ 키워드만 나열 (`병원 진료 코 귀 목 알레르기 안산 본오동`)
- ✅ 자연스러운 한 문장으로 "이 페이지가 뭔지" 설명

## 5단계 — URL 정비

기본 URL은 `imweb.me/20` 같은 숫자입니다. **의미 있는 영문 단어**로 바꾸면 검색·공유 모두 유리.

### 설정 방법

메뉴 설정 창에서 **URL** 항목을 `notice`, `about`, `service` 같은 영문으로 변경.

### URL 작성 원칙

- 영문 소문자 + 하이픈 (`our-team`, NOT `Our_Team`)
- 짧고 의미 있게
- 한글 URL 가능하지만 SNS 공유 시 인코딩되어 보기 안 좋음
- 한 번 정한 URL은 **바꾸지 않기** (기존 검색 결과 깨짐)

## 6단계 — 이미지 ALT 텍스트 & 대표 이미지

### 이미지 ALT

검색엔진은 이미지를 직접 인식하지 못합니다. **ALT 텍스트**로 알려줘야 이미지 검색에 노출되고, 본문 키워드 가중치에도 반영됩니다.

위젯에 이미지 추가 시 **이미지 설정** 창에서 **대체 텍스트**(ALT) 입력.

### 대표 이미지 (Open Graph)

카카오톡·페이스북 등에 링크 공유될 때 표시되는 이미지.

- 관리자 → **환경설정 > 일반 > 대표 이미지**.
- **PNG, 1200×630px** 권장.
- 페이지별로도 메뉴 설정에서 **메뉴 대표이미지** 따로 지정 가능.

![대표 이미지 설정 (Open Graph)](/imweb/372534e6e3e7f.png)

## 검색 노출 확인하기

### 1주차

- 네이버 검색창에 `site:mycompany.com` 입력 → 색인된 페이지 보임.
- 안 보이면 아직 수집 안 됨. 더 기다리세요.

### 2~4주차

- 네이버·구글에 사이트명 검색 → 본 사이트가 상단에 떠야 정상.
- 안 나오면: 메타 정보 다시 점검, SSL 적용 여부 확인, robots.txt 차단 여부 확인.

### 검색 안 되는 흔한 원인

- ❌ **SSL 미적용** — 네이버는 https 미적용 사이트 색인 우선순위 낮음.
- ❌ **로봇 차단** — `환경설정 > SEO > 고급 설정`에서 검색엔진 수집이 꺼져있는지 확인.
- ❌ **빈 사이트** — 페이지가 1~2개뿐이거나 콘텐츠가 너무 적으면 색인 안 됨.

## ⚠️ 자주 막히는 포인트

- **사이트 설명을 비워둠** → 검색 결과에서 본문 첫 문장이 그대로 노출되어 어색.
- **모든 페이지 제목을 사이트명으로** → 페이지마다 다른 제목 필수.
- **이미지 ALT를 안 넣음** → 이미지 검색 트래픽 손실.
- **2주도 안 기다리고 "왜 검색 안 돼요?"** → 최소 2~4주는 기다리세요.
- **검색 결과를 사이트 본인이 확인** → 본인은 항상 자기 사이트가 위에 떠요(개인화 검색). **시크릿창**에서 확인.

## ❓ FAQ

**Q. 등록만 하면 바로 1등으로 떠요?**
A. 아닙니다. SEO 등록은 "검색되도록 만드는" 1차 작업이고, 상위 노출은 콘텐츠 양·품질·외부 링크·체류 시간 등 수십 가지 요인이 결합된 결과입니다. 등록 후 **꾸준한 콘텐츠 업데이트**가 핵심.

**Q. 네이버 검색광고(파워링크)와 SEO는 다른 건가요?**
A. 완전히 다릅니다. 파워링크는 유료 광고(클릭당 과금), SEO는 자연 검색 결과입니다. 파워링크를 해도 SEO 순위는 오르지 않아요. 둘 다 별도로 챙겨야 합니다.

**Q. AI 검색(ChatGPT, Perplexity)에도 노출하려면?**
A. 별도 대응이 필요합니다. [AI 검색(AEO/GEO) 대응하기](#) 가이드 참고.

**Q. 메타 키워드가 정말 중요하지 않나요?**
A. 거의 영향 없습니다. 구글은 2009년부터 메타 키워드를 무시하고, 네이버도 비중이 매우 작습니다. 시간 들이지 마세요.

**Q. 도메인 옮기면 검색 순위 떨어지나요?**
A. 일시적으로 떨어집니다. 301 리다이렉트로 기존 URL의 권한을 새 URL로 넘기면 회복됩니다. 도메인 이전 전에 반드시 계획 수립이 필요해요.
