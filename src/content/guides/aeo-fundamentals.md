> **AEO**(Answer Engine Optimization)·**GEO**(Generative Engine Optimization)는 ChatGPT·Perplexity·Claude·네이버 큐: 같은 AI 답변 엔진이 **우리 사이트를 출처로 인용하도록** 만드는 새로운 최적화입니다. 기존 SEO가 "검색 결과 1페이지에 노출"이 목표였다면, AEO는 "AI 답변 안에 인용"이 목표예요.

## 검색이 바뀌고 있다

### 전통 검색

```
사용자 검색  →  10개 링크 결과  →  사용자가 클릭  →  사이트 방문
```

### AI 검색

```
사용자 질문  →  AI가 여러 사이트 종합  →  답변 + 출처 표시
                                          ↑
                                    여기에 우리 사이트 노출이 목표
```

### 통계로 보는 변화

- 2024년 미국 18~34세의 **31%가 AI 검색을 일주일에 한 번 이상 사용** (Gartner)
- ChatGPT 검색 기능(SearchGPT) 출시 후 **구글 검색 점유율 1% 하락** (Statcounter)
- 한국: 네이버 큐: 베타 서비스, 구글 SGE 한국어 지원 시작 (2024 하반기)

> 💡 AI 검색이 기존 검색을 완전히 대체하지는 않지만, **트래픽 일부 잠식 + 새로운 노출 채널** 동시 진행. 양쪽 다 준비해야.

## AEO와 GEO 차이

용어가 혼용되지만 구분하면:

| 항목 | AEO (Answer Engine) | GEO (Generative Engine) |
|---|---|---|
| 대상 엔진 | 정답 도출형 (Alexa, Siri, Google AI Overview) | 생성형 답변 (ChatGPT, Claude, Perplexity) |
| 답변 형태 | 짧은 정답 1개 | 긴 종합 답변 + 다중 출처 |
| 최적화 핵심 | 명확한 단일 답변 문장 | 신뢰 신호·다양한 인용 가능성 |
| 등장 시기 | 2015~ (음성 검색 시대) | 2023~ (LLM 검색 시대) |

실무에서는 **둘을 합쳐 "AEO"로 통칭**하는 경우가 많음. 본 가이드도 그 관행 따름.

## 왜 지금 준비해야 하나

### 1. 선점 효과

AEO는 초기 단계. 경쟁자가 적습니다. **지금 잘 만들어두면 6~12개월 뒤 AI가 우리 사이트를 우선 학습·인용**.

### 2. 트래픽 잠식 대비

AI 답변에 만족한 사용자는 클릭하지 않을 수 있음. 클릭이 줄어도 **출처 표시되면 브랜드 인지도 ↑**, 결국 직접 검색 유입으로 회수.

### 3. B2B·전문 분야 강점

AI 답변은 **신뢰성·전문성**을 우선 인용. 의료·법무·전문 서비스는 AEO가 광고보다 효과적.

### 4. 기존 SEO 자산 활용

SEO 잘 되어 있으면 AEO도 90% 됨. 추가 작업이 적어 **ROI 높음**.

## AI는 어떤 사이트를 인용하나

대형 AI 모델들의 공통 선호 신호:

### 1. 명확한 정답 문장

페이지 상단에 "이 페이지가 답하는 질문"의 **명확한 답변 한 문장**.

❌ "안녕하세요. 저희는 1990년부터…"
✅ "안산 본오동 ○○이비인후과는 코·귀·목 전문 진료를 제공합니다."

### 2. 구조화된 정보

표·리스트·FAQ·step-by-step. **AI가 파싱하기 쉬운** 구조.

❌ 줄글로만 가득
✅ 헤딩별 짧은 답변 + 표로 비교

### 3. 신뢰 신호

- 명확한 운영 주체 (회사명·연락처·주소)
- 작성자 정보 (E-E-A-T)
- 최종 업데이트 날짜
- 출처·인용 표기

### 4. 사용자 경험

- 빠른 로딩
- 모바일 친화
- 광고·팝업 최소
- HTTPS

### 5. 외부 인용

다른 사이트가 우리를 출처로 인용 → AI가 더 신뢰. 백링크와 같은 원리.

## AEO 5가지 핵심 작업

### 1. 페이지 상단 핵심 요약 (40~60단어)

페이지 첫 단락에 **그 페이지의 한 줄 답변** 명시.

예: "안산 본오동 ○○이비인후과는 코·귀·목 전문 진료를 제공합니다. 평일 야간 9시까지·토요일도 진료. 수면다원검사·알레르기 비염 전문."

### 2. FAQ 섹션 + FAQPage Schema

각 페이지에 **자주 묻는 질문 5~10개** 추가 + JSON-LD 구조화 데이터.

[FAQ schema 추가하기](/guides/aeo-faq-schema/) 상세.

### 3. llms.txt 작성

AI 크롤러 전용 표준 파일. 사이트 핵심 페이지를 명시.

[llms.txt 작성법](/guides/aeo-llms-txt/) 상세.

### 4. 헤딩을 질문 형태로

```
❌ 진료 시간
✅ 야간·토요일 진료는 가능한가요?
```

AI는 헤딩을 질문으로 인식하고 그 아래 답변을 인용 후보로 봄.

### 5. 신뢰 신호 강화

- 사이트 푸터에 사업자 정보·주소·연락처
- About 페이지에 운영 주체·연혁
- 블로그 글에 작성자 이름·날짜
- 의료·법무는 자격 명시

## AI 엔진별 특성

### ChatGPT (OpenAI)

- 학습 데이터 기준일 있음 (모델별 다름)
- 검색 기능(SearchGPT)으로 실시간 보강
- **신뢰 도메인 가중치 큼**: 정부·언론·전문 사이트
- 한국어 처리 우수

### Perplexity

- 항상 출처 명시 + 다중 인용
- **실시간 웹 검색 강함**
- 답변 길이 짧음 → 명확한 한 줄 답변 중요
- 한국어 점차 강화 중

### Claude (Anthropic)

- 학습 데이터 + 사용자 문서 분석
- 검색 통합도 진행 중
- **긴 형식·전문 분야 강함**
- 한국어 강함

### 네이버 큐:

- 베타 서비스 (2024 출시)
- **네이버 자체 콘텐츠 우선**: 블로그·카페·뉴스
- 일반 웹 출처는 한정적
- 향후 확대 예상

### 구글 AI Overview (SGE)

- 검색 결과 상단에 AI 요약 박스
- **기존 SEO 강한 사이트가 인용**됨
- 클릭률 감소 우려 ↔ 인용 시 노출 효과
- 한국어 지원 진행 중

## 측정하기

AEO 효과 측정은 SEO보다 까다로움. 직접 추적:

1. **수동 테스트**: 월 1회 본인 업종 관련 질문을 ChatGPT·Perplexity에 던져 우리 사이트 인용 확인
2. **브랜드 검색량 추이**: AEO 잘되면 브랜드 직접 검색 ↑ (네이버 키워드 도구·GSC)
3. **참조 트래픽**: GA4에서 referral 트래픽 중 chatgpt.com, perplexity.ai 등 확인
4. **상위 답변 영역 노출**: Google AI Overview 박스에 인용 여부

## ⚠️ 주의할 점

- **AI 답변 만족 = 클릭 안 함**: 단기적으로 트래픽 ↓ 가능. 브랜드 인지가 진짜 가치.
- **AI가 잘못 인용**: 우리 사이트 정보를 다르게 요약·왜곡 가능. 정기 모니터링 필요.
- **너무 광고스러운 카피는 인용 안 됨**: 정보성·객관성이 핵심.
- **이미지·동영상만 있는 페이지는 인용 불가**: AI는 텍스트 중심. 텍스트 필수.

## ❓ FAQ

**Q. AEO는 SEO 다 한 다음에 해도 되나요?**
A. **동시에 하는 게 좋음**. 사이트 새로 만들 때 처음부터 AEO 친화적으로 작성하면 추가 작업이 거의 없습니다. 기존 사이트라면 SEO 먼저 안정시킨 후 AEO 보강.

**Q. 광고를 안 돌리는데도 AEO 해야 하나요?**
A. AEO는 광고가 아닌 자연 노출 영역. **광고 안 돌릴수록 AEO가 더 중요**.

**Q. 한국 사이트인데 ChatGPT·Perplexity 신경 써야 하나?**
A. 사용자가 점점 늘고 있어요. 특히 **20~40대 전문직·자영업자** 사이에서 AI 검색 빈도 ↑. 무시할 수 없는 채널.

**Q. AI가 답변에 인용한 걸 어떻게 확인하나요?**
A. 직접 ChatGPT·Perplexity에서 본인 업종 질문 → 답변 하단 출처 확인. Perplexity가 가장 명확.

**Q. 작은 비즈니스 사이트인데 AI에 노출 가능한가요?**
A. **로컬 비즈니스는 오히려 유리**. "안산 본오동 이비인후과" 같은 구체적 검색에 대형 사이트가 없어 작은 로컬 사이트가 인용됨.
