> 가비아·카페24·후이즈 등에서 산 도메인을 아임웹에 연결하려면 **네임서버를 아임웹이 발급한 4개 주소로 변경**하면 됩니다. 변경 후 최대 48시간 이내에 사이트가 보입니다. 아임웹에서 직접 산 도메인은 추가 작업 없이 바로 연결됩니다.

## 시작하기 전에

- **요금제 확인**: 개인 도메인 연결은 Starter 이상 요금제부터 가능합니다.
- **준비물**: 도메인 구입처 로그인 정보(가비아·카페24·후이즈 등), 아임웹 관리자 권한.
- **소요 시간**: 작업 5~10분 + 네트워크 전파 최대 48시간.

---

## 전체 흐름 (3단계)

```
1. 아임웹에 도메인 추가  →  2. 할당된 네임서버 확인  →  3. 도메인 구입처에서 네임서버 변경
```

---

## 1단계 — 아임웹에 도메인 추가하기

1. 관리자 페이지 접속 → 좌측 메뉴에서 **환경설정 > 도메인** 클릭.

![아임웹 관리자 환경설정 > 도메인 메뉴 위치](/imweb/ce2c393282f11.png)

2. 우측 상단 **도메인 구입/연결** 버튼 클릭.
3. **타사 구매 도메인 연결**(또는 **내 도메인 연결하기**)을 선택.
4. 입력창에 도메인을 입력합니다. **`http://`·`www`는 빼고** 본 도메인만 입력 (예: `mycompany.com`).

![도메인 입력 화면 — http와 www 제외하고 본 도메인만](/imweb/6bcbff1ead1b8.png)

5. **연결** 버튼을 누르면 도메인이 목록에 추가됩니다.

> 📌 이 단계만 끝나도 도메인이 추가된 것처럼 보이지만, **네임서버 변경(3단계)을 완료해야 실제로 접속됩니다.**

## 2단계 — 할당된 네임서버 확인

1. 추가한 도메인 옆 **수정하기**(또는 **관리하기**) 버튼 클릭.
2. **할당된 네임서버 1차 ~ 4차** 주소를 메모장에 복사해 둡니다.

![아임웹이 발급한 네임서버 1~4차 주소](/imweb/2f8ca7f4e9751.png)

> ⚠️ 네임서버는 도메인마다 달라요. 다른 사이트에서 본 예시 주소를 그대로 쓰지 마세요. **반드시 본인 도메인에 표시된 주소**를 사용해야 합니다.

## 3단계 — 도메인 구입처에서 네임서버 변경

구입처에 따라 메뉴 위치가 다릅니다. 자주 쓰는 3곳만 정리했어요.

### 가비아

1. [가비아 도메인 관리](https://dns.gabia.com) 로그인.
2. 좌측 메뉴 **도메인 정보 변경** → 변경할 도메인 선택.
3. 상단 **네임서버** 탭 클릭.
4. 기존 네임서버를 모두 삭제하고, 2단계에서 메모한 1~4차 네임서버를 입력.
5. **소유자 인증** → **적용** 버튼.

![가비아 네임서버 변경 화면](/imweb/7ce7be3cdbcf5.png)

### 카페24

1. [카페24 도메인 센터](https://hosting.cafe24.com/) 로그인.
2. **도메인 정보 변경 > 네임서버 변경** 클릭.
3. 대상 도메인 선택 → **네임서버 변경하기**.
4. 휴대폰/이메일 본인 인증.
5. **다른 네임서버** 옵션 선택 → 1~4차 주소 입력 → **IP 확인**(필수) → **변경하기**.

![카페24 네임서버 변경 화면](/imweb/a0a0b2ab60b38.png)

### 후이즈

1. 후이즈 마이페이지 로그인.
2. **도메인 관리 > 네임서버 변경**.
3. 대상 도메인 선택, 네임서버 입력 후 저장.


> 위 3곳 외(호스팅케이알, Wix 등)는 동일한 원리예요. **"네임서버 변경" 메뉴를 찾아 1~4차 주소를 입력**하면 됩니다. 메뉴 위치가 안 보이면 해당 호스팅사 고객센터에 "네임서버 변경 요청"이라고 문의해도 처리해 줍니다.

## 연결 확인하는 법

- 변경 후 **5분 ~ 48시간** 사이에 사이트가 새 도메인으로 접속됩니다.
- 빠른 확인이 필요하면 **시크릿 브라우저 창**에서 도메인을 입력해 보세요(브라우저 캐시 영향 최소화).
- `www` 붙은 주소(`www.mycompany.com`)와 안 붙은 주소(`mycompany.com`) **둘 다 확인**.

## SSL 자동 적용 여부

도메인 연결이 완료되면 아임웹에서 **SSL 신청** 메뉴가 활성화됩니다. 회사·병원 사이트는 SSL이 사실상 필수예요 — 회원 가입·문의 폼·결제가 없어도 검색엔진 노출과 신뢰도에 직결됩니다. [imweb-ssl-install](/dev/imweb/imweb-ssl-install/) 참고.

## 메일도 같이 쓰고 싶다면

`name@mycompany.com` 같은 이메일을 운영하려면 **MX 레코드 설정**이 추가로 필요합니다. 구글 워크스페이스·다음 스마트워크·네이버웍스 등 메일 서비스의 MX 정보를 받아 아임웹 도메인 설정에 등록하면 됩니다. [imweb-mx-setup](/dev/imweb/imweb-mx-setup/) 참고.

---

## ⚠️ 자주 막히는 포인트

- **`http://`나 `www`를 같이 입력해서 인식 안 됨** → 본 도메인만 입력.
- **네임서버를 1차만 입력하고 끝냄** → 4개 모두 입력해야 안정적.
- **변경 후 바로 접속 시도** → 최소 5분, 길면 48시간 기다리세요.
- **여러 도메인을 한 아임웹 사이트에 연결 후 한 주소로 모으고 싶을 때** → 대표 도메인 자동 연결 설정 활용 ([imweb-primary-domain](/dev/imweb/imweb-primary-domain/)).

## ❓ FAQ

**Q. 네임서버 변경 후 며칠이 지나도 접속이 안 돼요.**
A. 도메인 구입처에 입력한 네임서버 주소가 1글자라도 틀리면 작동하지 않습니다. **아임웹에 표시된 4개 주소를 다시 복사해 정확히 일치하는지** 확인하세요. 그래도 안 되면 시크릿 창에서 시도해 캐시 문제인지 확인합니다.

**Q. 아임웹에서 직접 도메인을 사도 되나요?**
A. 가능합니다. 아임웹에서 사면 네임서버 변경 단계가 생략돼 가장 간편해요. 다만 가비아·카페24보다 가격이 조금 비싸고, 추후 다른 호스팅으로 옮길 때 절차가 한 단계 더 필요합니다.

**Q. 한 도메인을 여러 아임웹 사이트에 연결할 수 있나요?**
A. 1차 도메인(`mycompany.com`)은 한 사이트에만 연결됩니다. 다만 2차 도메인(`shop.mycompany.com`, `notice.mycompany.com`)은 여러 사이트에 각각 연결할 수 있어요. [imweb-subdomain](/dev/imweb/imweb-subdomain/) 참고.

**Q. 한글 도메인도 연결되나요?**
A. 됩니다. 다만 이메일 호환성·검색엔진 친화도 면에서 영문 도메인을 권장합니다.

**Q. CNAME으로 연결하라고 들었는데요?**
A. 일반 사용자에게는 권장하지 않습니다. CNAME은 1차 도메인 메일이 막히는 등 제약이 큽니다. **아임웹 네임서버 방식이 가장 안정적**이에요. CNAME이 꼭 필요한 특수 케이스는 [imweb-cname](/dev/imweb/imweb-cname/) 참고.
