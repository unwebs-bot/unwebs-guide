> 병원 예약 알림, 주문 처리 알림, 입력폼 접수 알림 등 자동 메시지를 보내려면 **발신번호 등록 + 템플릿 검수** 두 단계가 필요합니다. 이메일과 달리 카카오 알림톡·SMS는 사전 승인이 필수.

## 시작하기 전에

- **발신용 휴대전화 또는 회사 번호** 필요 (소유자 명의).
- **사업자**라면 사업자등록증, **개인**이라면 신분증 준비.
- **소요 시간**: 발신번호 등록 5~7 영업일 + 알림톡 템플릿 1~2 영업일.
- **충전 필요**: SMS·알림톡 모두 발송 건당 과금. 사전 충전.

## SMS·알림톡·LMS·MMS 차이

| 종류 | 글자수 | 비용 | 특징 |
|---|---|---|---|
| **SMS** | 90바이트 (한글 45자) | 약 9원/건 | 가장 저렴, 어디든 전송 |
| **LMS** | 2,000바이트 (한글 1,000자) | 약 28원/건 | 장문 가능 |
| **MMS** | 2,000바이트 + 이미지 | 약 50원/건 | 이미지 첨부 |
| **알림톡** | 1,000자 | 약 8~10원/건 | 카톡으로 발송, 사전 검수 필요 |
| **브랜드 메시지** (구 친구톡) | 1,000자 | 약 13원/건 | 카카오 채널 친구만, 광고 가능 |

> 💡 **알림톡이 가장 효율적**: 가격은 SMS와 비슷하지만 카카오톡 도달률·읽음률이 훨씬 높음.

## 발신번호 등록 (전제 조건)

거짓 발신번호로 인한 피해 예방을 위해 **사전 등록된 번호로만** 발송 가능. 통신사 규제.

### 1단계: 발신번호 등록 신청

1. 관리자 → **사용자 관리 > 자동 메일·SMS·알림톡 > 발신번호 등록**.
2. **신청** 버튼.
3. 단계별 입력:
   - 사업자 여부와 명의자 유형 선택
   - 발신번호 (휴대폰 또는 일반 전화)

### 2단계: 서류 제출

명의자 유형에 따라 필요 서류:

| 유형 | 필요 서류 |
|---|---|
| **개인 명의** | 신분증 사본 + 휴대폰 가입증명원 |
| **사업자 본인 명의** | 사업자등록증 + 명의자 신분증 |
| **사업자, 다른 명의 번호** | 사업자등록증 + 번호 소유자 신분증 + **위임장** |
| **운영자가 등록 시** | 사업자 등록증 + 명의자 위임장 |

> ⚠️ **명의가 일치하지 않으면 거부됩니다.** 사업장 대표 명의 휴대폰을 발신번호로 쓰는 게 가장 간단.

### 3단계: 심사 대기

- 영업일 기준 **3~7일** 소요.
- 결과는 신청자 휴대폰으로 **SMS 알림**.

![발신번호 등록 단계](/imweb/b9f3b79b579c4.png)

### 4단계: 등록 완료 후

- SMS 발송 즉시 가능.
- 카카오 알림톡은 **추가로 템플릿 검수** 필요 (다음 섹션).

## 카카오 알림톡 시작하기

### 사전 요구사항

- **비즈니스로 등록된 카카오톡 채널**.
- **카카오 알림톡 부가서비스 가입** (유료).
- 발신번호 등록 완료.

### 카카오톡 채널 만들기

1. [카카오 비즈니스](https://business.kakao.com) 접속.
2. **카카오톡 채널 만들기** → 사업자 정보 입력.
3. 비즈니스 인증 (사업자등록증 업로드).
4. 채널 ID 확인 (예: `@○○이비인후과`).

### 아임웹 부가서비스 가입

1. 관리자 → **부가서비스** → **알림톡** → **신청**.
2. 카카오톡 채널 ID 입력 → 연동.

## 알림톡 템플릿 만들기

알림톡은 **광고 메시지 발송 금지**. **정보성·고객 액션에 대한 응답**만 가능.

### 템플릿 작성

1. 관리자 → **자동 메일·SMS·알림톡 > 알림톡 템플릿**.
2. **+ 새 템플릿** → 종류 선택:
   - 회원가입 알림
   - 주문 알림
   - 배송 알림
   - 입력폼 접수 알림 등
3. 본문 작성 (변수 활용 가능):

   ```
   {이름}님,
   {사이트명} 진료 예약이 정상 접수되었습니다.

   - 예약 일시: {예약일시}
   - 진료과: {진료과}

   변경·취소는 {연락처}로 연락주세요.
   ```

4. **검수 요청**.

### 검수 통과 기준

✅ **통과**:
- 고객이 직접 요청한 정보 (예약 확인, 주문 알림)
- 객관적 사실 안내

❌ **거부**:
- "이벤트 안내" 같은 광고성 문구
- "할인 중", "특가" 등 마케팅 표현
- 메시지가 너무 짧거나 정보 부족

### 검수 소요 시간

영업일 기준 **1~2일**. 거부 시 사유 통지받음 → 수정 후 재제출.

> 💡 광고성 메시지는 **브랜드 메시지(구 친구톡)**로 보내야 합니다. 알림톡과 다른 채널.

## SMS·알림톡 발송 방법

### 1. 자동 발송 (이벤트 트리거)

- **회원가입 시** → 자동 알림톡
- **입력폼 제출 시** → 자동 SMS 또는 알림톡
- **주문 처리 시** → 자동 알림톡

설정: 자동 메일·SMS·알림톡 → 이벤트별 활성화.

### 2. 수동 발송 (캠페인)

- 관리자 → **사용자 관리 > 메시지 보내기**.
- 대상 회원 그룹 선택 → 템플릿 선택 → 발송.

> ⚠️ **광고성 수동 발송**은 야간(20:00~08:00) 금지. 알림톡은 24시간 가능.

## ⚠️ 자주 막히는 포인트

- **명의가 다른 번호로 발신번호 등록** → 거부됨. 명의 통일하거나 위임장 제출.
- **알림톡에 광고 문구 포함** → 검수 거부. 정보성으로만.
- **템플릿 변수 오타** (`{이름}` → `{이름`) → 발송 실패 또는 변수 그대로 표시.
- **잔액 부족** → 발송 안 됨. **자동 충전** 설정 권장.
- **카카오 알림톡 발송했는데 SMS로 와짐** → 수신자가 카카오톡 미사용 또는 차단 상태. 알림톡은 자동으로 SMS로 fallback.

## ❓ FAQ

**Q. 알림톡 발송 후 응답을 받을 수 있나요?**
A. 알림톡은 일방향. **버튼 추가**로 사이트·전화 이동은 가능. 양방향 대화는 카카오톡 채널 상담 사용.

**Q. 발신번호 등록 신청을 했는데 거부됐어요.**
A. 가장 흔한 사유는 명의 불일치. 명의자 신분증 + 가입증명원·통신사 확인서를 다시 제출.

**Q. 알림톡과 SMS를 둘 다 보낼 수 있나요?**
A. 가능합니다. 우선순위 설정: 알림톡 발송 시도 → 실패하면 SMS로 자동 fallback (아임웹 기본 동작).

**Q. 글자 수가 SMS 한계를 넘으면?**
A. 자동으로 LMS로 전환되며 비용이 SMS의 3배가량. 글자수 짧게 유지하는 게 비용 효율적.

**Q. 발송 통계는 어디서 보나요?**
A. 관리자 → **사용자 관리 > 통계 > 메시지 발송**. 발송 건수·성공/실패·잔액 확인.
