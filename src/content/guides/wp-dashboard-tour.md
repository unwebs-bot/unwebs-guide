> WordPress 관리자(`/wp-admin`)에 처음 들어가면 좌측 메뉴가 14개나 됩니다. 다 알 필요는 없고 **자주 쓰는 5~6개**만 익히면 충분해요. 메뉴별 의미·우선순위·자주 쓰는 항목을 정리합니다.

## 대시보드 진입

```
https://mycompany.com/wp-admin
```

ID·비밀번호 로그인 → 대시보드 표시.

## 좌측 메뉴 14가지 — 우선순위

| 메뉴 | 우선순위 | 자주 쓰나? |
|---|---|---|
| 알림판 (Dashboard) | ⭐ | 시작 페이지 |
| **글** | ⭐⭐⭐ | 매일 |
| 미디어 | ⭐⭐ | 자주 |
| **페이지** | ⭐⭐⭐ | 자주 |
| 댓글 | ⭐⭐ | 댓글 운영 시 |
| **외모 (Appearance)** | ⭐⭐ | 초기 + 가끔 |
| **플러그인** | ⭐⭐ | 초기 + 가끔 |
| 사용자 (Users) | ⭐ | 가끔 |
| 도구 (Tools) | ⭐ | 가끔 |
| **설정 (Settings)** | ⭐⭐ | 초기 + 가끔 |

플러그인 설치 시 추가되는 메뉴:
- SEO 플러그인 (Rank Math·Yoast)
- 폼 플러그인 (Contact Form 7)
- 보안 플러그인 (Wordfence)
- 백업 플러그인 (UpdraftPlus)
- 등

## 1. 알림판 (Dashboard)

로그인 후 첫 화면. 사이트 상태 한눈에.

### 주요 위젯

- **사이트 헬스(Site Health)**: WordPress 권장 설정 점수
- **활동(Activity)**: 최근 글·댓글
- **빠른 초안(Quick Draft)**: 빠르게 글 메모

### 활용 팁

- "사이트 헬스 점수" 정기 확인 — 보안·성능 권장사항
- 활동에 모르는 글·댓글 보이면 즉시 점검 (해킹 신호 가능)

## 2. 글 (Posts) — 매일 사용

블로그·뉴스성 콘텐츠.

### 하위 메뉴

- **모든 글**: 발행 완료·임시저장 글 목록
- **새 글 추가**: 새 글 작성
- **카테고리**: 글 분류
- **태그**: 글 키워드 표기

### 자주 쓰는 작업

- 새 글 작성 → 카테고리·태그·대표 이미지 설정 → 발행
- 옛 글 일괄 편집 (목록에서 체크 → 일괄 작업)
- 카테고리 정리·통합

## 3. 미디어 (Media)

이미지·동영상·파일 저장소.

### 자주 쓰는 작업

- 라이브러리에 이미지 업로드
- 이미지 정보 (제목·캡션·**대체 텍스트**) 입력
- 옛 미디어 정리 (사용 안 한 이미지 삭제)

> 💡 **대체 텍스트(Alt Text) 필수**. SEO와 접근성 양쪽.

## 4. 페이지 (Pages) — 자주 사용

About·Contact·Service 같은 정적 페이지.

### 글 vs 페이지 차이

- **글**: 날짜순 정렬 (블로그·뉴스)
- **페이지**: 정적, 카테고리 없음 (회사 소개·서비스·문의)

자세한 차이는 [글 vs 페이지](/guides/wp-post-vs-page/) 참고.

## 5. 댓글 (Comments)

방문자 댓글 관리.

### 자주 쓰는 작업

- **승인**: 댓글 공개
- **스팸**: 스팸 처리 (학습됨)
- **휴지통**: 부적절 댓글

### 댓글 안 받기

`설정 > 토론`에서 "새 글에서 댓글 허용" 체크 해제 → 새 글부터 댓글 비활성.

## 6. 외모 (Appearance) — 초기 작업

테마·디자인 영역.

### 하위 메뉴

- **테마**: 설치된 테마 + 새로 설치
- **사용자 정의(Customize)**: 라이브 미리보기로 디자인 변경
- **위젯**: 사이드바·푸터 위젯
- **메뉴**: 사이트 메뉴 구조
- **테마 파일 편집기**: PHP·CSS 직접 편집 (위험)

### 자주 쓰는 작업

- 새 테마 설치·전환
- 사용자 정의에서 색상·로고·메뉴 설정
- 위젯 배치

⚠️ **테마 파일 편집기**는 권장하지 않음 (실수로 사이트 다운). [자식 테마](/guides/wp-child-theme/) 사용.

## 7. 플러그인 (Plugins) — 가끔 사용

기능 확장.

### 하위 메뉴

- **설치된 플러그인**: 활성·비활성 관리
- **새로 추가**: 공식 저장소 검색·설치
- **플러그인 파일 편집기**: PHP 직접 편집 (위험)

### 자주 쓰는 작업

- 새 플러그인 설치·활성화
- 플러그인 업데이트 (월 1회)
- 안 쓰는 플러그인 비활성화·삭제

## 8. 사용자 (Users) — 가끔 사용

관리자·운영자·기고자 관리.

### 역할 (Role)

- **관리자(Administrator)**: 모든 권한
- **편집자(Editor)**: 모든 글·페이지 편집
- **저자(Author)**: 본인 글만 발행
- **기고자(Contributor)**: 본인 글 작성, 발행은 X
- **구독자(Subscriber)**: 댓글만

[사용자·권한 관리](/guides/wp-user-permissions/) 참고.

## 9. 도구 (Tools) — 가끔 사용

가져오기·내보내기·사이트 헬스.

### 자주 쓰는 작업

- **내보내기**: 글·페이지 XML 백업
- **가져오기**: 다른 사이트에서 데이터 이전
- **사이트 헬스**: 권장 설정 점검

## 10. 설정 (Settings) — 초기 + 가끔

사이트 전체 설정.

### 하위 메뉴

- **일반**: 사이트 제목·태그라인·시간대
- **쓰기**: 기본 카테고리·기본 글 형식
- **읽기**: 첫 페이지 표시 방식·검색엔진 색인 허용
- **토론**: 댓글 정책
- **미디어**: 이미지 자동 크기
- **고유주소(Permalinks)**: URL 구조 (**가장 중요**)
- **개인정보(Privacy)**: 개인정보 처리방침 페이지

### 가장 먼저 설정해야 할 것

- 시간대 (Asia/Seoul)
- 고유주소 → **게시물 이름** 선택
- 검색엔진 색인 허용 (개발 중일 때만 차단)

## 빠른 접근 — 단축키

| 화면 | 단축키 |
|---|---|
| 새 글 | `/wp-admin/post-new.php` |
| 새 페이지 | `/wp-admin/post-new.php?post_type=page` |
| 글 목록 | `/wp-admin/edit.php` |
| 미디어 | `/wp-admin/upload.php` |
| 플러그인 | `/wp-admin/plugins.php` |
| 설정 | `/wp-admin/options-general.php` |

브라우저 즐겨찾기에 등록하면 빨라요.

## 자주 받는 질문

### Q. 메뉴가 너무 많아 보기 어려워요.

**Screen Options(화면 옵션)** — 우상단 탭. 메뉴별로 표시할 항목·열 선택 가능.

### Q. 한국어로 안 보이는데?

`설정 > 일반 > 사이트 언어`에서 **한국어** 선택. WordPress 코어 한글팩 자동 다운로드.

### Q. 모바일에서 관리자 접속 가능?

가능하지만 UX는 PC 권장. 모바일 전용 앱 **WordPress for iOS/Android**도 있음.

### Q. 플러그인 설치하면 새 메뉴가 생기는데 어디서 관리?

좌측 메뉴 어딘가 또는 **설정 → [플러그인명]** 하위에 추가됨. 플러그인별로 위치 다름.

### Q. 화면 색 어둡게(다크 모드) 가능?

WordPress 기본 X. 플러그인 **WP Admin Theme**·**Dashboard Color Schemes** 활용.

## ⚠️ 자주 하는 실수

- **테마 파일 편집기에서 코드 수정 후 사이트 다운**: 자식 테마 사용
- **모든 플러그인 활성화 유지**: 안 쓰는 건 비활성화·삭제 (보안·속도)
- **댓글 승인 안 함**: 스팸 폭주. 자동 승인 또는 모더레이션 필수
- **사이트 헬스 무시**: 보안·성능 권장사항 정기 점검

## ❓ FAQ

**Q. 관리자 비밀번호 잊었어요.**
A. 로그인 페이지에서 "비밀번호 잊으셨나요?" 클릭 → 가입 이메일로 재설정 링크. 이메일 못 받으면 [흰화면·관리자 접속 불가](/guides/wp-white-screen-debug/) 참고.

**Q. 대시보드가 갑자기 흰 화면이에요.**
A. 플러그인·테마 충돌 가능성. [흰화면 디버그](/guides/wp-white-screen-debug/) 참고.

**Q. 영문 메뉴 vs 한글 메뉴, 어느 게 표준?**
A. 한글이 권장. 다만 영문 자료가 많아 학습 시 영문도 알아두면 좋음. 본 가이드는 한글 표기.

**Q. 사용자 관리에서 운영진을 추가했는데 안 보여요.**
A. 가입 메일 발송됐는지 확인. 운영진은 자기 메일 인증 후 활성화. 또는 `wp-admin/users.php`에서 직접 추가.

**Q. 어떤 플러그인이 설치돼 있는지 한 번에 보고 싶어요.**
A. 플러그인 > 설치된 플러그인 → 우상단 **Screen Options**에서 페이지당 200개로 변경하면 전체 한눈에.
