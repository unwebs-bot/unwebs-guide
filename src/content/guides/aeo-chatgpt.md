> 사람들이 점점 ChatGPT·Perplexity·Claude로 질문하고 있어요. 이 AI들이 답변할 때 **우리 사이트를 출처로 인용하도록** 만드는 게 AEO의 최종 목표. 엔진별 인용 패턴 차이와 실전 노출 전략을 정리합니다.

## AI 답변 엔진별 차이 (2026년 5월 기준)

| 엔진 | 출처 표시 | 인용 패턴 | 한국어 강도 |
|---|---|---|---|
| **ChatGPT (SearchGPT)** | 답변 안에 링크 | 1~3개 출처, 통합형 답변 | 강함 |
| **Perplexity** | 답변 + 사이드바에 출처 목록 | 5~10개 출처 명시 | 중간 (개선 중) |
| **Claude** | 답변 + 검색 모드에서 출처 | 2~5개, 정확성 우선 | 강함 |
| **네이버 큐:** | 답변 + 출처 카드 | 네이버 자체 콘텐츠 우선 | 매우 강함 |
| **구글 AI Overview** | 검색 결과 상단 박스 | 3~5개 출처, 기존 SEO 강한 사이트 | 강화 중 |

각 엔진을 동시 공략. 단일 엔진에만 집중하지 말 것.

## AI가 인용하는 사이트의 공통 특징

여러 AI 엔진의 인용 패턴 관찰 결과:

### 1. 구조화된 답변이 본문 상단

페이지 첫 단락에 **"이 페이지가 답하는 질문의 답"**. AI가 가장 먼저 읽는 영역.

### 2. 명확한 사실·수치

추상적 표현보다 구체적 수치·날짜·이름.

❌ "빠른 응답을 자랑합니다"
✅ "평균 응답 시간 2시간 이내 (평일 09~21시)"

### 3. FAQ 또는 Q&A 형식

질문 형태의 헤딩 + 짧고 완결된 답변. AI가 가장 좋아하는 구조.

### 4. 출처·근거 명시

"근거: ~ 통계", "출처: ~ 발표" 같은 외부 신뢰 신호.

### 5. 최신성

문서 하단에 "최종 업데이트: 2026-05" 표기. AI는 옛 정보 회피 경향.

### 6. 모호하지 않은 운영 주체

About 페이지·푸터에 운영자·연락처·주소. **YMYL 콘텐츠**(의료·법무·금융)는 자격까지.

## ChatGPT (SearchGPT) 노출 전략

ChatGPT의 검색 기능은 **실시간 웹 검색 + 모델 지식**의 하이브리드.

### 인용되는 사이트 특성

- 도메인 권위 ↑ (백링크·트래픽 누적)
- 정보형 콘텐츠 (광고 카피 X)
- 빠른 LCP·HTTPS·모바일 친화
- 위키피디아·정부·언론·전문 도메인 우선

### 전략

1. **콘텐츠 깊이 우선** — 1500~3000자 이상의 정보형 글
2. **Top 10 / 비교 형식** — "X가지 방법", "A vs B" 같은 구조화된 답변
3. **인용 가능한 통계** — 구체적 수치 (출처 표기 시 더 좋음)
4. **외부 인용 자산** — 우리 콘텐츠를 다른 사이트가 인용해주면 인용 신호 ↑

## Perplexity 노출 전략

Perplexity는 **다중 출처 + 출처 명시**가 특징. 5~10개 사이트를 답변 사이드바에 노출.

### 인용되는 사이트 특성

- 실시간 색인된 최신 콘텐츠
- 명확한 답변 1~3문장
- 단일 주제 집중 페이지
- 다양한 출처와 함께 인용되어도 차별점이 있음

### 전략

1. **롱테일 키워드** 공략 — Perplexity는 대형 사이트가 없는 niche에서 작은 사이트도 자주 인용
2. **답변형 콘텐츠** — "OO은 무엇인가요?", "OO하는 법" 같은 의문문 헤딩
3. **출처 인용** — 본문에 다른 신뢰 출처 인용 (Perplexity가 학습)
4. **빠른 색인** — 새 콘텐츠 발행 후 Search Console 색인 요청

## Claude·기타 LLM 노출 전략

Claude·LLM 일반의 학습 데이터에는 시점이 있음. 모델 출시 후 학습 데이터는 고정되지만 검색 모드에서는 실시간 보강.

### 전략

1. **장기 자산화** — 시간이 지나도 가치 있는 콘텐츠 (Evergreen)
2. **백링크 + 외부 언급** — 학습 데이터에 자주 등장하는 사이트가 우선 인용
3. **전문성 명시** — E-E-A-T 신호 강화

## 네이버 큐: 노출 (한국 시장)

네이버의 자체 AI 답변. 한국 사용자에게 점점 노출 비중 ↑.

### 인용되는 사이트 특성

- 네이버 자체 플랫폼 (블로그·카페·뉴스) 강력 우선
- 외부 웹사이트는 한정적 인용
- 사이트 신뢰도·SSL·메타 정보 영향

### 전략

1. **네이버 블로그 운영** — 핵심 콘텐츠를 사이트 + 블로그 양쪽에
2. **서치어드바이저 등록** — [네이버 SEO 가이드](/guides/seo-naver-deep/) 참고
3. **모바일 최적화** — 큐: 검색은 모바일 비중 큼

## 구글 AI Overview 노출 (SGE)

구글 검색 결과 상단의 AI 박스. 한국어 점진 지원 중.

### 인용되는 사이트 특성

- 기존 SEO 강한 사이트 (구글 검색 1페이지 등 진입)
- 짧고 명확한 답변
- 구조화 데이터 적용
- E-E-A-T 강함

### 전략

기존 SEO 잘 되어 있으면 자동으로 후보. 추가:
1. **FAQ schema** ([FAQ schema 가이드](/guides/aeo-faq-schema/))
2. **첫 단락 정답형** 작성
3. **헤딩 질문 형태**

## 실전 — 한 페이지에 모든 AEO 적용

`편한이비인후과` 진료 안내 페이지 예:

```markdown
# 안산 본오동 편한이비인후과 진료 안내

> 코·귀·목 전문 진료를 평일 야간 9시까지·토요일 오후 1시까지 제공합니다.
> 수면다원검사·알레르기 비염 클리닉·중이염을 주로 다룹니다.

## 진료 종목

### 코 진료
알레르기 비염·만성 비염·축농증·코골이 진단·수면다원검사를 시행합니다.
보험 적용 진료 95% 이상.

### 귀 진료
중이염·외이도염·돌발성 난청·이명·청력검사 가능. 성인·소아 모두 진료.

### 목 진료
인후염·편도염·성대 결절·갑상선 초음파 검사. 음성 직업군 진료 다수.

## 진료 시간

- 평일: 09:00 ~ 21:00
- 토요일: 09:00 ~ 13:00
- 일요일 / 공휴일: 휴진

## 자주 묻는 질문

### 야간 진료는 몇 시까지 가능한가요?
평일 저녁 9시까지 진료. 토요일은 오후 1시까지.

### 예약 없이 방문해도 진료받을 수 있나요?
워크인 진료 가능. 다만 대기 시간이 길 수 있어 전화 또는 카카오톡 예약 권장.

### 수면다원검사는 보험 적용되나요?
코골이·수면무호흡 진단 목적은 건강보험 적용. 본인 부담금 10~15만원.

---

**위치**: 경기 안산시 상록구 본오동 OOO
**전화**: 031-XXX-XXXX
**카카오톡**: 채널 @편한이비인후과

최종 업데이트: 2026-05
```

이 페이지는 **모든 AI 엔진에 인용 가능**한 구조.

## 모니터링 방법

### 월 1회 — 인용 확인

본인 업종 관련 질문 10개를 4개 엔진에 던지기:

```
ChatGPT/Perplexity/Claude/큐:에 동일 질문
"안산 본오동에 야간 진료 가능한 이비인후과 있나요?"
"수면다원검사 보험 적용 되나요?"
"안산 알레르기 비염 전문 병원 추천"
...
```

답변에 우리 사이트 인용 여부 기록 → 트래킹.

### GA4에서 referral 트래픽

GA4 > 획득 > 트래픽 획득 > 세션 소스/매체 → `chatgpt.com`, `perplexity.ai` 등 확인.

### Search Console — AI Overview 노출

구글 Search Console > 검색 모양 (Search appearance)에서 AI Overview 노출 확인 (한국 지원 진행 중).

## ⚠️ 자주 하는 실수

- **광고성 카피로 도배**: AI는 정보성 우선. 광고만 있으면 인용 X
- **이미지·인포그래픽만**: AI는 텍스트 중심. 텍스트 필수
- **답변 없이 클릭 유도**: "자세한 건 클릭" → AI는 클릭 못 함. 답변 본문에 명시
- **여러 주제 한 페이지**: 단일 주제·단일 의도 페이지가 인용 ↑
- **출처·근거 없는 주장**: "최고", "1위" 같은 검증 안 된 표현 → 신뢰 ↓

## ❓ FAQ

**Q. AEO 효과 보려면 얼마나 걸려요?**
A. 콘텐츠 보강·llms.txt 적용 후 **2~3개월** 안에 일부 인용 시작. 안정적 인용까지 6~12개월.

**Q. 작은 사이트인데 ChatGPT가 우리를 인용할까요?**
A. **로컬·niche 키워드는 가능**. 일반 키워드는 대형 사이트와 경쟁이라 어려움. 구체적 롱테일 위주 공략.

**Q. AI에 인용되면 트래픽이 늘어나요?**
A. 즉각 효과는 작을 수 있음 (AI 답변에 만족한 사용자는 클릭 안 함). 다만 **브랜드 인지 + 신뢰도 ↑**로 장기적으로 직접 검색·문의 증가.

**Q. ChatGPT가 우리 사이트를 잘못 요약해요.**
A. 본문에 정확한 정보를 분명히 명시. 모호한 표현·다중 해석 가능 표현 제거. 시간이 지나면 자동 보정되는 경향.

**Q. AI에 인용 안 되게 하려면?**
A. `robots.txt`에 AI 봇 차단:
```
User-agent: GPTBot
Disallow: /

User-agent: ClaudeBot
Disallow: /

User-agent: PerplexityBot
Disallow: /
```
다만 트래픽 손해 우려. 일반적으로 권장하지 않음.
